# companies_analysis_project


есть две папки:

\base_results
\experimental_results



 \base_results  -  содержит базовые, проверенные и устойчивые результаты.
Это минимально необходимый набор данных и вычислений, подтверждённый и воспроизводимый.
Папка служит в качестве эталона для сравнения с дальнейшими улучшениями и экспериментами.

Внутри находятся:
- корректные и стабильные значения;
- результаты, прошедшие проверку;
- данные, которые можно использовать как основу для отчётов и финальных выводов.

Применялись сайты:
- https://companies.rbc.ru/
- https://marketing-tech.ru/






\experimental_results  -  Содержит расширенные, тестовые и исследовательские результаты.
Эта директория используется для экспериментов, гипотез, альтернативных подходов и промежуточных вычислений, которые могут давать более широкий или более глубокий вывод, но не гарантируются как финальные.

Внутри находятся:
- экспериментальные модели, вычисления и графики;
- альтернативные методики и их результаты;
- материалы для сравнения и дальнейшего развития проекта.
